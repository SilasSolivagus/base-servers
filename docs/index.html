五轴保证

agent 的权限,被五个轴向锁死

给 agent 一把 API key,就是交出一份永久、越权、不可撤销、不可审计的凭证。base-servers 给它的是另一种东西。

限范围

只允许授权时点名的动作。

限时

短命,自动过期。

可撤销

kill-switch,秒级生效。

可归因

每步都追溯到 agent 和它背后的人。

≤ 授权人

永不超过授权它的人自己拥有的权限。
